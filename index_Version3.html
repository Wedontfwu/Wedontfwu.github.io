Timeline

A simple, easy-to-edit timeline. Edit timeline.json or use the form to add items.

Add timeline item (saved locally)

How to add permanently

Edit timeline.json in the repo (on your computer or in the GitHub web editor). Use ISO dates like 2026-02-04.

Fields: date, title, subtitle, content, image.

Publishing

Push to GitHub and enable Pages in repository Settings (select branch main and root). The site will be available at https://<your-username>.github.io/<repo>/.
